# SwirlX Website - Professional Heat Exchanger Marketing Site

## Project Overview
A modern, professional website for SwirlX heat exchangers, featuring smooth animations, responsive design, and lead generation capabilities. Built with React, Tailwind CSS, and advanced animation libraries.

---

## Key Features
- **Modern Design** - Professional industrial aesthetic
- **Responsive Layout** - Perfect on all devices
- **Smooth Animations** - Framer Motion + GSAP integration
- **Fast Performance** - Optimized for speed and SEO
- **Lead Generation** - Contact forms and call-to-actions
- **Global Reach** - Deployed on Vercel with CDN

---

## Technology Stack
- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS + Custom CSS
- **Animations:** Framer Motion + GSAP ScrollTrigger
- **Deployment:** Vercel (serverless)
- **Domain:** swirlx-engineering.com

---

## Quick Start

### Prerequisites
- Node.js 16+ and npm
- Git for version control

### Installation
```bash
# Clone the repository
git clone https://github.com/hugodeengineering/swirlx-website.git
cd swirlx-website

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to Vercel
vercel --prod
```

### Development Commands
```bash
npm run dev          # Start development server (localhost:5173)
npm run build        # Build for production
npm run preview      # Preview production build
vercel --prod        # Deploy to production
```
